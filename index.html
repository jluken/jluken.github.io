Downloading and Installing Maven

Requirements

JDK 1.7 or newer
10MB for Maven and up to 500MB for repository

Downloads

Download either the .zip or tar.gz files from this site:
https://maven.apache.org/download.cgi
Download binary for easy installation, or source code if you want to modify and compile it yourself

Installing Maven from file

Extract the files

unzip apache-maven-3.3.9-bin.zip
or
tar xzvf apache-maven-3.3.9-bin.tar.gz

Add /bin folder to PATH

Windows:
Open "System Properties">"Advanced">"Environment Variables" and add PATH variable
C:\Program Files\apache-maven-3.3.9\bin
Unix:
export PATH=/opt/apache-maven-3.3.9/bin:$PATH

Installing Maven via terminal

Alternatively, Maven can be directly installed to Unix Systems through terminal commands


Ubuntu:

sudo apt-get install maven

OSX via Homebrew:

brew install maven
