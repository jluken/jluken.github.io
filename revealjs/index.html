Downloading and Installing Maven

Configuring an existing project with Maven

  1. Clone the project.
  2. Remove any existing binaries or packages.
  3. Create the Maven `pom.xml` configuration file.

Configuring an existing project with Maven

Clone the project:

						
git clone https://github.com/overben/Snake.git

Remove any existing binaries or packages:

						
cd Snake
rm SnakeGame.jar

Configuring an existing project with Maven

Create a basic Maven pom.xml configuration file:

						
<project
  xmlns="http://maven.apache.org/POM/4.0.0"
  xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
  xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd" >
  <modelVersion>4.0.0</modelVersion>

  <groupId>com.github.overben.snake</groupId>
  <artifactId>snake</artifactId>
  <version>1.0</version>
  <packaging>jar</packaging>

  <name>Snake</name>
  <url>https://github.com/overben/Snake</url>
</project>

Configure Maven to build Java sources that are not in the standard package-based location: 

						
<build>
  <sourceDirectory>src</sourceDirectory>
  <plugins>
    <plugin>
      <groupId>org.apache.maven.plugins</groupId>
      <artifactId>maven-jar-plugin</artifactId>
      <version>3.0.2</version>
      <configuration>
        <archive>
          <manifest>
            <addClasspath>true</addClasspath>
            <classpathPrefix>lib/</classpathPrefix>
            <mainClass>Main</mainClass>
          </manifest>
        </archive>
      </configuration>
    </plugin>
  </plugins>
</build>

Building the project with Maven

						
cd Snake
mvn package

Running Unit Tests with Maven

Add junit tests to projdir/src/test/java

Run tests using: 

mvn test

Add the following lines to your pom.xml for the junit dependencies


<dependencies>
	<dependency>
		<groupId>junit</groupId>
		<artifactId>junit</artifactId>
		<version>4.8.2</version>
		<scope>test<scope>
	</dependency>
</dependencies>
