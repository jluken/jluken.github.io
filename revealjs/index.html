Downloading and Installing Maven

Requirements

JDK 1.7 or newer
10MB for Maven and up to 500MB for repository

Downloads

Download either the .zip or tar.gz files from this site:
https://maven.apache.org/download.cgi
Download binary for easy installation, or source code if you want to modify and compile it yourself

Installing Maven from file

Extract the files

unzip apache-maven-3.3.9-bin.zip
or 
tar xzvf apache-maven-3.3.9-bin.tar.gz

Add /bin folder to PATH in Windows:

Open "System Properties">"Advanced">"Environment Variables" and add PATH variable
C:\Program Files\apache-maven-3.3.9\bin

Add /bin folder to PATH in Unix:

export PATH=/opt/apache-maven-3.3.9/bin:$PATH

Installing Maven via terminal

Alternatively, Maven can be directly installed to Unix Systems through terminal commands


Ubuntu:

sudo apt-get install maven


OSX via Homebrew:

brew install maven

Configuring an existing project with Maven

Clone the project:

						
git clone https://github.com/overben/Snake.git

Remove any existing binaries or packages:

						
cd Snake
rm SnakeGame.jar

Configuring an existing project with Maven

Create a basic Maven pom.xml configuration file:

						
<project
  xmlns="http://maven.apache.org/POM/4.0.0"
  xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
  xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd" >
  <modelVersion>4.0.0</modelVersion>

  <groupId>com.github.overben.snake</groupId>
  <artifactId>snake</artifactId>
  <version>1.0</version>
  <packaging>jar</packaging>

  <name>Snake</name>
  <url>https://github.com/overben/Snake</url>
</project>

Configure Maven to build Java sources that are not in the standard package-based location: 

						
<build>
  <sourceDirectory>src</sourceDirectory>
  <plugins>
    <plugin>
      <groupId>org.apache.maven.plugins</groupId>
      <artifactId>maven-jar-plugin</artifactId>
      <version>3.0.2</version>
      <configuration>
        <archive>
          <manifest>
            <addClasspath>true</addClasspath>
            <classpathPrefix>lib/</classpathPrefix>
            <mainClass>Main</mainClass>
          </manifest>
        </archive>
      </configuration>
    </plugin>
  </plugins>
</build>

Building the project with Maven

						
cd Snake
mvn package

Running Unit Tests with Maven

Add junit tests to projdir/src/test/java

Run tests using: 

mvn test

Add the following lines to your pom.xml for the junit dependencies

					
<dependencies>
	<dependency>
		<groupId>junit</groupId>
		<artifactId>junit</artifactId>
		<version>4.8.2</version>
		<scope>test<scope>
	</dependency>
</dependencies>

Example Junit Test
